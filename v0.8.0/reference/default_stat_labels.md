# Defaults for Statistical Arguments

Returns a named list of statistics labels

## Usage

``` r
default_stat_labels()
```

## Value

named list

## Examples

``` r
# stat labels
default_stat_labels()
#> $mean
#> [1] "Mean"
#> 
#> $sd
#> [1] "SD"
#> 
#> $var
#> [1] "Variance"
#> 
#> $median
#> [1] "Median"
#> 
#> $p25
#> [1] "Q1"
#> 
#> $p75
#> [1] "Q3"
#> 
#> $min
#> [1] "Min"
#> 
#> $max
#> [1] "Max"
#> 
#> $n
#> [1] "n"
#> 
#> $N
#> [1] "N"
#> 
#> $p
#> [1] "%"
#> 
#> $n_cum
#> [1] "Cumulative n"
#> 
#> $p_cum
#> [1] "Cumulative %"
#> 
#> $N_obs

#> $N_nonmiss
#> [1] "N Non-missing"
```

# cards: Analysis Results Data

Construct CDISC (Clinical Data Interchange Standards Consortium)
compliant Analysis Results Data objects. These objects are used and
re-used to construct summary tables, visualizations, and written
reports. The package also exports utilities for working with these
objects and creating new Analysis Results Data objects.
